# 📋 Sistema de Cadastro de Nomes em Java

Um mini sistema feito em Java que permite adicionar, visualizar e remover nomes utilizando uma matriz 3x3.

## 🚀 Funcionalidades

* ✅ Adicionar nomes
* 📄 Listar nomes cadastrados
* ❌ Remover nomes
* ⚠️ Sistema de erros inspirado em códigos HTTP (404, 400, 500)

## 🧠 Conceitos utilizados

* Matrizes (arrays bidimensionais)
* Estruturas de repetição (for)
* Condicionais (if / switch)
* Tratamento de erros (try/catch)
* Entrada de dados com Scanner

## 💻 Exemplo

```bash
===== Menu =====
[ 1 ] - Adicionar nome
[ 2 ] - Ver nome(s)
[ 3 ] - Remover nome
[ 4 ] - Sair
```
